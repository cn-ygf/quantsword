QuantSword
===========

一个使用electron开发的web shell管理工具

本代码还处于开发中...请勿使用！

编译与运行
------

编译

    git clone https://github.com/cn-ygf/quantsword
    cd quantsword
    cnpm install

运行

    cnpm start

打包

    cnpm run build
